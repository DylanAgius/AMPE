\noindent{\bf \sc Description}

\begin{alltt}
N_VL1Norm returns the L1 norm of x

   Usage:  RET = N_VL1Norm ( X [, COMM] )

If COMM is not present, N_VL1Norm returns the L1 norm of 
the local portion of X. Otherwise, it returns the global
L1 norm..
\end{alltt}
