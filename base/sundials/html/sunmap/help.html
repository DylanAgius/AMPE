Usage


  1. Supported browsers

    The map has been successfully tested on the following browsers (Javascript must be enabled and the browser must accept cookies):
    • Firefox v.1.5.0.3
    • IE v.6.0
    • Mozilla 1.7
    • Opera v.8.54


  2. Naviagtion

    All usual Google map navigatino features are available (map dragging, zoom, etc.)
    Hovering over a marker will display the user's name in a tooltip and the user's location (latitude and longitude) in the message area below the map.
    Clicking on a marker will display the user's information (name, city and country, and, if provided a photo).
    A listing of all markers can be toggle on and off form the "Show List"/"Hide List" button.
    Clicking on the "zoom" icon next to a name in the list will zoom and center the map around the location of that user.
    You can delete the marker that you own (identified through a cookie set at creation time) by clicking on the "cross" icon in the last column.


  3. Adding a marker

    Adding a marker with your information consists of two steps.

    After clicking the "Add" button, you must first indicate the location on the map. A new temporary (green) marker will be displayed. You can continue changing its location by clicking on the map in a different location until you submit the information or cancel the addition of a new marker.
    Zooming, dragging, and panning the map continue to be active, so you can be precise in specifying your location. However, in "Add" mode, displaying the information window for existing markers is disabled.

    Once you are satisfied with the location of the new marker, fill in the "Add" form, by specifying your name, city, country, and (optionally) a link to a photo. This information will be displayed in an information window associated with the new marker and in the listing table.
    Click the "Submit" button to store the information for the new marker. The map will reload and the newly created marker will be shown on the map and in the listing.
    You can click the "Cancel add" button at any time to discard the temporary marker and return to navigation mode.

    Note that you can "own" only one marker. If you attempt to add a second one, you will be prompted to replace the existing one.
