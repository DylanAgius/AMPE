Latest SUNDIALS release (November 2006)

What's new in the latest release?


Users that are interested in more than one of the SUNDIALS solvers are encouraged to download the entire suite at the same time.
This reduces the risk of using incompatible versions of the solvers and shared modules.

The sundialsTB Matlab toolbox is now included in sundials-2.3.0.tar.gz and is not distributed separately anymore.
